### ８章　関数型インターフェース、ラムダ式

ラムダ式は下記のような文法
* 関数型インターフェースの型 変数名 = (引数) -> {処理};
* カッコも省略することができる。ただし、引数が一つの時だけ。
* 引数は型を書かなくてもいい

関数型インターフェースは、利用用途に分けて使う。
* Consumer：消費者
* Supplier：供給者
* Predicate：断定
* Function :処理

Predicate は何らかの処理の結果としてboolean型の値を返すための関数型インターフェース
