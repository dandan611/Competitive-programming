## メモ

* main()メソッド
  * Javaアプリケーションの実行時にJVM(Java Virtual Machine)によって呼び出されるアプリケーションを起動するためのメソッド。
  * Javaアプリケーションには必ず1つのmain()メソッドが定義されている必要がある。
  *javaプログラムは大文字と小文字を区別する
  * 疑問:なんで'public'なのか。
  * 疑問:なんで'static'なのか。
  
* JVM
  * プログラムの実行やメモリ管理を行うソフトウェア、JREで提供される。

* Java EE
  * 大規模システム構築用のエディション
